# Git Commands Used and Learned

git status
# Shows the current status of the working directory and staging area, including changes to be committed, untracked files, and branch information.

git add <filename>
# Adds a specific file to the staging area, preparing it to be included in the next commit.

git add .
# Adds all changed files (new, modified, deleted) in the current directory and subdirectories to the staging area.

git commit -m "commit message"
# Commits the staged changes to the local repository with a descriptive commit message.

git push origin main
# Pushes the local main branch commits to the remote repository named 'origin'.
